#### type( )
``` python
# função type( ) mostra o tipo de dado
type(5)
>>> int
type("Python")
>>> str
type(3.11)
>>> float
print(type(500))
>>> int
print(type("Python"))
>>> str
```
#### Textos (Strings)
``` python
print("Python")
>>> Python
print("2026")
>>> 2026
print("Python " + "2026") # concatenar duas "string" com o sinal de "+" / "espaço" também é um caracter
>>> Python 2026
print("2026" * 3) # multiplica o valor de string com o sinal de "*"
>>> 202620262026
```
